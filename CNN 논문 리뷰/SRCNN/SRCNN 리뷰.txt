저해상도 -> 고해상도    :  이미지 확대 --> 100 -> 300 : 빈공간 발생 -> bicubic interpolation : 부드럽게 확대
저해상도 이미지(100x100)는 먼저 bicubic interpolation을 통해 원하는 출력 크기로 확대(300x300)되며, 이 과정은 학습 대상이 아니라 전처리 단계로 처리된다. 이후 bicubic으로 생성된 입력 이미지 Y와 원본 고해상도 이미지 X를 사용하여 CNN이 end-to-end 지도학습을 수행한다. 이때 네트워크의 중간 레이어에서는 비선형성을 위해 ReLU가 사용된다.

SRCNN은 bicubic interpolation으로 미리 확대된 저해상도 이미지 Y를 입력으로 받아, 이를 정답 고해상도 이미지 X에 가깝게 복원하는 매핑 F를 학습한다. 저자들은 이 매핑을 세 단계로 설명하는데, 첫째는 패치 추출 및 표현, 둘째는 비선형 매핑, 셋째는 복원 단계이다. 또한 기존의 patch representation이 사실상 convolution 연산으로 해석될 수 있음을 보이며, 이를 학습 가능한 CNN 구조로 통합하였다.

Figure 2에서 SRCNN은 세 단계로 구성된다. 첫 번째 레이어는 bicubic-upsampled 입력 이미지로부터 패치 단위의 feature map을 추출하고, 두 번째 레이어는 이를 고해상도 패치 표현으로 비선형 변환한다. 마지막 레이어는 주변 공간 정보를 결합하여 최종 고해상도 이미지를 복원한다. 특히 두 번째 레이어는 1×1 convolution을 통해 저해상도 특징 표현을 고해상도 표현으로 매핑하는 역할을 수행한다. 두 번째 레이어는 새로운 공간 패턴을 추출하는 단계가 아니라, 첫 번째 레이어가 이미 추출한 국소 패치 특징을 고해상도 패치 표현으로 변환하는 단계이다. 따라서 공간적 이웃을 다시 넓게 볼 필요보다, 각 위치의 채널 특징 벡터를 다른 특징 공간으로 매핑하는 것이 중요하므로 1×1 convolution이 사용된다.

SRCNN의 마지막 레이어는 두 번째 레이어에서 생성된 고해상도 패치 표현들을 실제 이미지 공간으로 복원하는 역할을 수행한다. 저자들은 기존 sparse-coding 기반 SR에서 사용되던 patch aggregation 과정을 convolution 연산으로 해석하고, 이를 학습 가능한 마지막 복원층으로 통합하였다. 이에 따라 패치 추출, 비선형 매핑, 복원이라는 세 단계가 모두 convolutional layer 형태로 정식화되며, SRCNN은 이 전체 과정을 end-to-end로 최적화하는 구조를 갖는다.







초해상도(super-resolution)는 저해상도 이미지를 더 큰 해상도의 이미지로 복원하는 문제이다. 저해상도 이미지를 고해상도로 변환하려면 출력 이미지의 크기가 커져야 하므로, 결과적으로 더 많은 픽셀 위치가 필요해진다. 따라서 핵심 과제는 단순히 이미지를 확대하는 것에 그치지 않고, 새롭게 필요한 픽셀 위치마다 원본 고해상도 이미지에 가까운 값을 얼마나 정확하게 추정하느냐에 있다.

기존 초해상도 방법은 patch extraction, dictionary/manifold 기반 표현, reconstruction, aggregation 등 여러 단계로 구성되며, 각 단계를 분리하여 처리하기 때문에 전체 파이프라인을 end-to-end로 최적화하기 어렵다는 한계를 가진다. SRCNN은 이러한 복잡한 SR 파이프라인을 CNN 관점에서 재해석하고, 이를 하나의 학습 가능한 네트워크로 통합한 모델이다.

SRCNN은 bicubic interpolation으로 미리 확대된 저해상도 이미지 Y를 입력으로 받아, 이를 원본 고해상도 이미지 
X에 가깝게 복원하는 매핑 F(Y)를 end-to-end로 학습한다. 실험 결과, SRCNN은 bicubic interpolation 및 sparse-coding 기반 방법보다 더 높은 PSNR을 보였고, CPU 환경에서도 비교적 빠른 속도를 달성하였다.

저자들은 본 논문의 기여를 세 가지로 정리한다. 첫째, 초해상도 문제를 CNN 기반의 end-to-end 학습 문제로 정식화하였다. 둘째, 제안 방법과 전통적인 sparse-coding 기반 SR 방법 사이의 관계를 정립하여 네트워크 구조 설계의 근거를 제시하였다. 셋째, 딥러닝이 초해상도 문제에서 복원 품질과 속도 측면 모두에서 효과적임을 입증하였다.

기존 초해상도 연구의 주류는 patch-based 방식으로, 저해상도 패치와 고해상도 패치 사이의 관계를 학습하는 접근이었다. 이 과정에서는 dictionary, manifold, sparse coding 등의 표현 방식이 주로 사용되었다. 그러나 patch extraction과 aggregation은 별도의 전처리/후처리로 다루어졌고, 전체 파이프라인을 end-to-end로 최적화하지 못했다는 한계가 있었다.

SRCNN의 입력은 원래의 저해상도 이미지가 아니라 bicubic interpolation을 통해 원하는 출력 크기로 확대된 이미지 Y이다. 이 보간 과정은 확대된 해상도에서 새롭게 필요한 픽셀 위치의 값을 주변 픽셀을 참고해 부드럽게 추정하는 전처리 단계이며, 학습 대상은 아니다. 이후 CNN은 Y를 입력으로 받아 원본 고해상도 이미지 X에 가까운 결과를 생성하도록 지도학습된다. 이 과정에서 중간 레이어에는 비선형성을 위해 ReLU가 사용된다.

Figure 1은 SRCNN의 성능이 학습 초반부터 빠르게 향상되며, bicubic interpolation과 sparse-coding 기반 방법보다 더 높은 PSNR을 달성함을 보여준다. 또한 시각적 비교를 통해 SRCNN이 더 선명한 복원 결과를 제공함을 제시한다.

Figure 2는 SRCNN의 전체 구조를 보여준다. 첫 번째 레이어는 bicubic-upsampled 입력 이미지로부터 패치 단위의 feature map을 추출하고, 두 번째 레이어는 이를 고해상도 패치 표현으로 비선형 변환한다. 마지막 레이어는 주변 공간 정보를 결합하여 최종 고해상도 이미지를 복원한다. 특히 두 번째 레이어는 1×1 convolution을 사용하여, 첫 번째 레이어가 이미 추출한 국소적 특징을 채널 방향으로 재조합하고 이를 고해상도 패치 표현으로 매핑한다. 이는 구조를 단순하게 유지하면서도 충분한 복원 성능을 확보하기 위한 설계이다.

Figure 3은 기존 sparse-coding 기반 초해상도 방법을 CNN 관점에서 해석한 그림이다. 저자들은 patch extraction, non-linear mapping, reconstruction의 세 단계가 모두 convolutional layer 형태로 표현될 수 있음을 보이며, 특히 patch aggregation 과정 역시 convolution 연산으로 해석하여 학습 가능한 복원층으로 통합하였다. 이를 통해 SRCNN은 전체 SR 파이프라인을 end-to-end로 최적화하는 구조로 정식화된다.
